## 🛠️ Installation

### 1. Clone the repository

```bash
git clone <repository-url>
cd product-catal  og
```

### 2. Configure Frontend environment variables

```bash
cd product-catalog
cp .env.example .env
```


### 3. Install Frontend dependencies

```bash
npm install
```

### 4. Run in development mode

```bash
npm run dev
```

The app will be available at: **`http://localhost:3000`**
